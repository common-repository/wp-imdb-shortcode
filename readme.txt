== Description ==

**WP IMDB Shortcode** is get movies or actors/actresses content that for use need to use simple shortcode **imdb for wordpress**.

It can be **imdb wordpress automation plugin** to get imdb movie data and actors imdb data in **imdb wordpress**.

Example: If you want show Particular artist details **without imdb api** so you need to go one imdb website and code of title you need to copy and need to put in shortcode as well you can put on **imdb widget wordpress**.

It can be support any **imdb wordpress theme** to bring data in your website

**wordpress imdb movie plugin** for good choice to using without api plugin.

**Artist Shorcode**

    name = [wpoc_imdb name="nm0004874"]
    (director, actor, writer imdbname)


**Movie Shorcode**

    title = [wpoc_imdb title="tt0232500"]
    (movie imdbtitle)


**WP IMDB Shortcode** can be make without api can be access imdb data in [wordpress](https://en.wikipedia.org/wiki/WordPress).

You can add **WP IMDB** you can use shotcode for integration **imdb wordpress**.
